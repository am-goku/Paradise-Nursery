# Paradise Nursery

This is a simple React-based e-commerce application for browsing and purchasing houseplants. It features a landing page, product listings categorized by type, a shopping cart with quantity management, and basic navigation.

## Features
- Home landing page with company name and "Get Started" button.
- About Us page with company details.
- Product listing with categories, plant details, and "Add to Cart" functionality.
- Shopping cart displaying items, totals, quantity controls, and buttons for checkout/continue shopping.
- Uses React, React Router, Redux Toolkit, and CSS for styling.

## Setup
1. Clone the repo.
2. Run `npm install`.
3. Run `npm start`.
